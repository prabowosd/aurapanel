# AuraPanel Product Overview

## Mission

AuraPanel is a modern hosting control plane designed around decoupled architecture, deterministic operations, and zero-trust security direction.

## Core Value Propositions

- Control plane and serving plane separation
- Runtime-backed endpoint transparency
- Operationally honest responses (no fake success)
- Deterministic automation for repeatable operations
- Security-first behavior under production constraints

## Primary Runtime Domains

- Website provisioning and OpenLiteSpeed vhost alignment
- DNS lifecycle management
- SSL lifecycle management
- Mailbox and mail transport workflows
- Database provisioning and tuning
- Backup and recovery workflows
- Migration and import workflows

## Operating Principles

1. Performance-sensitive paths stay simple.
2. Reliability beats cosmetic feature count.
3. Security defaults should fail closed.
4. Every automation path should be observable.
5. Migratability matters for real-world adoption.

## Ideal Audience

- Hosting providers
- Multi-tenant infrastructure teams
- Agencies managing client hosting workloads
- Teams migrating from legacy or commercial panel stacks
