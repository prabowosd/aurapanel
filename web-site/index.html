FAQ

Frequently asked questions about AuraPanel hosting operations.

How do I install AuraPanel on a fresh server?

Use a fresh Ubuntu host with sudo/root privileges and run:

sudo apt update && sudo apt upgrade -y
curl -fsSL https://raw.githubusercontent.com/mkoyazilim/aurapanel/main/install.sh | sudo bash

After setup, open your panel hostname over HTTPS and login with the configured admin account.

Which ports are required for a standard deployment?

Open required network ports before acceptance tests:

  • Base access: 22, 80, 443
  • Panel and OLS operations: 7080, 8090
  • Mail stack if enabled: 25, 465, 587, 110, 995, 143, 993

If any required port is blocked, SSL, webmail or panel checks may fail.

How do I issue and verify Let's Encrypt SSL?

In Manage SSL, choose the website domain and issue certificate after DNS points to your server IP.

  • Verify OpenLiteSpeed is running before certificate reload.
  • Check `www` and root domain are mapped to the same vhost.
  • Retest with browser + `https` lock validation after issuance.
